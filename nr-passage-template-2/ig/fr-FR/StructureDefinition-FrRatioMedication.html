: Ratio with UCUM or EDQM codes if code is used

: https://interop.esante.gouv.fr/ig/fhir/eprescription/StructureDefinition/FrRatioMedication : 1.1.0-ballot
Draft : FrRatioMedication

Ratio with numerator and denominator unit UCUM or EDQM encoded if code is used

Ratio avec des unités UCUM ou EDQM si un code est utilisé.

Utilisations:

  • Utilise ce/t/te profil de type de données: FR Medication Non Compound, FR Medication and FR Medication Request

Vous pouvez également vérifier les usages dans le FHIR IG Statistics
